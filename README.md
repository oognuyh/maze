# **Maze Gnererator**
A maze generator with Hunt and kill algorithm using python.

## Features
  * Generate a maze.
  * Set starting and end points randomly. 
  
## Usage
  ```
  // if pygame is not installed
  pip3 install pygame

  python3 huntandkill.py
  ```
  
  * Generate
    * Just press the RETURN key
  
  * Color
     * Red : starting point
     * Green : end point
